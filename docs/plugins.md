Plugins will contain things such as Commands, Functions, Aliases, Auto
Completion Scripts, and more. Plugins are focused on enhancing your experience
in the shell.

# Enable Plugin
```
pms plugin enable [PLUGIN]
```

# Disable Plugin
```
pms plugin disable [PLUGIN]
```
