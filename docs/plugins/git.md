# git

## Requirements

* git

## Aliases

By enabling this plugin, you will gain access to these alias that you can use.

| Alias | Command       | Notes |
| ----- | ------------- | ----- |
| g     | git           |       |
| gd    | git diff      |       |
| gs    | git status    |       |
| gr    | git remote -v |       |
