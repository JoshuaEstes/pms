# zsh

Configures basic zsh shell behavior.

## Environment Variables

| Variable       | Default | Description                                   |
| -------------- | ------- | --------------------------------------------- |
| `DIRSTACKSIZE` | `10`    | Maximum size of the directory stack           |
| `KEYTIMEOUT`   | `1`     | Key binding timeout in tenths of a second     |
